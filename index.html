Memory Game

Introduction

"Memo" is a single player card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. If all pairs are matched within the given time period, game is won.

How to play?

  1. Click anywhere on the screen to proceed
  2. Press START button
  3. Click on TWO squares at a time to reveal contents and check for a match
  4. If there is a match the squares will remain flipped
  5. Successfully match all pairs in the table to win!
