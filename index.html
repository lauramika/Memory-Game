Memory Game

Introduction

It's a single player card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. If all pairs are matched within the given time period, game is won.

How to play?

  1. Press start button
  2. Click on TWO squares at a time to reveal contents and check for a match
  3. If there is a match the squares will remain flipped
  4. Successfully match all pairs in the table to win!
Flips: 0
Time: 60
